# Cloud_Computing_and_Devops
A document repository for my cloud computing and DevOps work, using my college assignments and exploring other AWS services. This is a remnant of the project folder I deleted for personal reasons. It serves as proof of work and demonstrates deployment processes.

Assignment 1: AWS EC2 instance deployment using a static HTML website.

Assignment 2: AWS S3 deployment using a static website. Implemented website policy on AWS, and deployed S3 using the console and AWS CLI.

Assignment 3: AWS RDS deployment to implement cloud DBs (SQL DBs) and worked on a CRUD app. For exploration, I changed my availability zone from Mumbai to Seoul, which resulted in some latency due to the geographical distance. It also led to a small charge, possibly due to varying service costs in different regions.
